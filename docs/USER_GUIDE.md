# Four Freedoms - User Guide

## Introduction

Four Freedoms is a web application that combines podcast playback with an interactive AI assistant. This guide will help you navigate the application and make the most of its features.

## Getting Started

### Accessing the Application

Open your web browser and navigate to the application URL. The application will load with the home page displaying featured podcasts and topics.

### Navigation

The application has several main areas:

1. **Header**: Contains the application title, search bar, and user authentication status.
2. **Topic Panel**: Located on the left side, displays topics you can follow.
3. **Content Panel**: The main area displaying podcasts and the chat interface.
4. **Bottom Bar**: Appears when a podcast is playing in minimized mode.

## Using the Application Without an Account

Without creating an account, you can:

1. Browse and search for podcasts
2. Listen to free podcasts
3. View topics
4. Use the chat assistant with basic functionality
5. Ask questions about podcasts

However, your preferences and chat history will not be saved, and you won't have access to premium content.

## Creating an Account

To access all features, create an account:

1. Click the "Sign In" button in the header
2. Select the "Sign Up" option
3. Fill in your details:
   - First name
   - Last name
   - Email address
   - Password
4. Click "Create account"

Alternatively, you can sign up using:
- Google account
- Facebook account
- Microsoft account

## Listening to Podcasts

### Finding Podcasts

1. **Browse Featured Podcasts**: The home page displays featured podcasts.
2. **Search for Podcasts**: Use the search bar in the header to find specific podcasts.
3. **Browse by Topic**: Select a topic from the left panel to see related podcasts.

### Playing a Podcast

1. Hover over a podcast card to reveal the play button
2. Click the play button to start playback
3. The podcast will appear in the chat interface and begin playing
4. Use the player controls to:
   - Play/pause
   - Adjust volume
   - Mute/unmute
   - Skip forward/backward
   - Track progress

### Minimizing the Player

When you scroll or navigate away from the podcast, it will minimize to the bottom bar. You can:

1. Continue listening while browsing other content
2. Click the maximize button to return to the full player view
3. Control playback from the minimized player

## Using the Chat Assistant

The chat assistant helps you interact with podcasts and get information.

### Asking Questions

1. Type your question in the input field at the bottom of the chat interface
2. Click "Ask" or press Enter to send your question
3. The assistant will process your question and provide a response

### Suggested Questions

When playing a podcast, the chat interface will display suggested questions related to the content. Click on any suggested question to ask it directly.

### Voice Input

To use voice input for questions:
1. Click the microphone icon next to the input field
2. Speak your question clearly
3. The question will be transcribed and sent to the assistant

## Managing Topics

### Browsing Topics

The left panel displays available topics. By default, it shows popular topics and those related to your location.

### Expanding the Topic Panel

1. Click on a topic to select it
2. The panel will expand to show more details about the topic
3. Related podcasts will appear in the content panel

### Following Topics

To follow a topic (requires an account):
1. Click the "+" button next to a podcast
2. The podcast's topics will be added to your followed topics
3. These topics will appear in your topic panel for easy access

## Account Management

### Profile Settings

To access your profile settings:
1. Click on your name/avatar in the header
2. Select "Profile" from the dropdown menu
3. Here you can update:
   - Personal information
   - Subscription details
   - Payment information

### Subscription Management

Four Freedoms offers three subscription tiers:

1. **Free**:
   - Basic podcast access
   - Limited chat functionality
   - Ad-supported experience

2. **Premium** ($9.99/month):
   - Full podcast library
   - Advanced chat features
   - Ad-free experience
   - Personalized recommendations

3. **Enterprise** ($29.99/month):
   - Everything in Premium
   - Team collaboration
   - API access
   - Priority support
   - Custom integrations

To upgrade your subscription:
1. Go to your profile
2. Select the "Subscription" tab
3. Choose your desired plan
4. Enter payment details
5. Confirm your subscription

## Accessibility Features

Four Freedoms includes several accessibility features:

1. **Keyboard Navigation**: All features can be accessed using keyboard shortcuts
2. **Screen Reader Support**: Compatible with popular screen readers
3. **Resizable Interface**: The chat panel can be resized by dragging the handle
4. **Text-to-Speech**: Podcast transcripts can be read aloud (Premium feature)
5. **Voice Input**: Questions can be asked using voice input

## Troubleshooting

### Common Issues

1. **Playback Issues**:
   - Check your internet connection
   - Try refreshing the page
   - Ensure your device's volume is turned up

2. **Chat Assistant Not Responding**:
   - Check your internet connection
   - Try asking a different question
   - Refresh the page

3. **Login Problems**:
   - Clear your browser cookies and try again
   - Reset your password if you've forgotten it
   - Contact support if issues persist

### Getting Help

If you encounter any issues not covered in this guide:
1. Check the FAQ section in the application
2. Contact support through the "Help" option in the user menu
3. Send an email to support@fourfreedoms.com
